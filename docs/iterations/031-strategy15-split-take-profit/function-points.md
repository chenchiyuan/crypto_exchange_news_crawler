## P0 功能点（MVP核心）

### FP-001: 快速止损机制
- **描述**: 实现3%快速止损
- **验收标准**: low <= buy_price × 0.97 时触发止损
- **依赖**: 无

### FP-002: 分批止盈逻辑
- **描述**: 实现5档分批止盈（2%, 3%, 4%, 5%, 6%）
- **验收标准**: 每档止盈20%仓位，达到价格条件时成交
- **依赖**: FP-003

### FP-003: 限价卖单管理
- **描述**: 每个持仓挂出5档限价卖单
- **验收标准**: high >= 挂单价格时成交
- **依赖**: 无

### FP-004: 持仓合并逻辑
- **描述**: 同价格持仓合并为单一订单
- **验收标准**: 合并后数量正确，止盈档位重新计算
- **依赖**: 无

### FP-005: 策略配置支持
- **描述**: 支持JSON配置止盈参数
- **验收标准**: tp_levels, first_tp_rate, tp_interval可配置
- **依赖**: 无
