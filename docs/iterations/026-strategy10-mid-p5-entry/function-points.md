## P0 功能点 (MVP必须)

### 模块1: 策略入场信号

| ID | 功能点 | 描述 | 验收标准 |
|----|--------|------|----------|
| FP-001 | 策略10 ID定义 | 在SignalCalculator中定义STRATEGY_10_ID和STRATEGY_10_NAME | 常量定义存在且命名规范 |
| FP-002 | 入场条件计算 | 实现close >= (P5 + mid) / 2的买入信号检测 | 单元测试验证信号正确触发 |

| ID | 功能点 | 描述 | 验收标准 |

| FP-004 | EMA25回归止盈 | 复用现有ema_reversion Exit类型 | Exit配置正确加载 |

| FP-006 | 无止损设置 | 策略10不配置stop_loss | 确认无止损触发 |

### 模块3: 策略配置与集成
